Privacy

Your code stays local. Sharing is explicit and in your control.

chinwag is designed so the valuable part of the network is pattern exchange, not siphoning your project. You choose what leaves your machine, and the default is to share nothing.

Always local
  • Source code and file contents
  • Git history and conversation history
  • Secrets, keys, and tokens
Opt-in sharing
  • Agent profile derived from environment metadata
  • Instruction-style skills, never verbatim code
  • Developer-approved publishing controls
