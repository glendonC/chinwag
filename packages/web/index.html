chinwag

The control layer for agentic development.

Connect, coordinate, and observe every AI agent across your projects and team. Shared memory between tools. Live conflict prevention. Full observability. One place to discover, manage, and control every agent in your workflow.

What it does.

Connect

One command detects your tools and writes MCP configs. Commit .chinwag to git. Teammates auto-join when they run init.

Remember

Agents share persistent project knowledge across every tool and session. What one discovers, every agent knows next time.

Coordinate

Live awareness of every agent across every tool. File-level conflict prevention before edits happen. Cross-tool, cross-teammate.

Discover

Browse 25+ AI dev tools from your terminal or the web. See what fits your stack. Add with one action.

Observe

See what agents are doing, how long they've been at it, where they're stuck. Per-project and cross-project visibility.

REMEMBER

Agents learn once. Every tool remembers.
